# Instalasi

Untuk mengaktifkan library Ideahut Spring Boot di project.

## Maven

``` xml
<dependency>
	<groupId>net.ideahut</groupId>
	<artifactId>ideahut-springboot-3x</artifactId>
	<version>26.04.04</version>
</dependency>
<dependency>
	<groupId>net.ideahut</groupId>
	<artifactId>ideahut-springboot-3.5x.1</artifactId>
	<version>26.04.04</version>
</dependency>

```

## Gradle

``` cmd
implementation group: 'net.ideahut', name: 'ideahut-springboot-3x', version: '26.04.04'
implementation group: 'net.ideahut', name: 'ideahut-springboot-3.5x.1', version: '26.04.04'
```
